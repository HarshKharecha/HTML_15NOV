🧠 Task 1: Create a Grid Container.

Question: How do you turn a normal container into a CSS Grid layout?

Description: Every grid starts by declaring display: grid on a parent container.

Hint: Use display: grid; in CSS.

🧠 Task 2: Define Two Equal Columns.

Question: Create two columns of equal width.

Description: Use grid-template-columns to define column structure.

Hint: Use 1fr 1fr for equal flexible columns.

🧠 Task 3: Create Three Fixed-Width Columns.

Question: Make a grid with 3 columns, each 100px wide.

Description: Fixed widths give consistent layout sizes.

Hint: Use pixel values.

🧠 Task 4: Create Mixed Column Sizes.

Question: Create a layout with one fixed column and two flexible ones.

Description: Mixing px and fr makes adaptable layouts.

Hint: Try 200px 1fr 2fr.

🧠 Task 5: Define Row Heights.

Question: Create two rows, one 100px and one 200px high.

Description: grid-template-rows defines row height.

Hint: Use pixel values for rows.

🧠 Task 6: Use repeat() for Columns.

Question: Make a grid with 4 equal columns using repeat().

Description: repeat() shortens repetitive patterns.

Hint: Use repeat(4, 1fr).

🧠 Task 7: Add Space Between Items.

Question: Add 20px of space between grid items.

Description: The gap property controls both row and column spacing.

Hint: Use gap: 20px;.

🧠 Task 8: Set Different Row and Column Gaps.

Question: Add 10px between rows and 30px between columns.

Description: row-gap and column-gap control spacing separately.

Hint: Use both properties or shorthand.

🧠 Task 9: Create a Two-Row, Three-Column Grid.

Question: Make a grid with 2 rows and 3 columns.

Description: Combine both row and column templates.

Hint: Use grid-template-rows and grid-template-columns.

🧠 Task 10: Use fr Units for Responsive Columns.

Question: Create 3 columns that share available space equally.

Description: fr units adapt automatically to screen size.

🧠 Task 11: Automatically Fit Columns.

Question: Make a responsive grid that adjusts the number of columns automatically.

Description: Use auto-fit and minmax() to make layouts that resize gracefully.

Hint: Use repeat(auto-fit, minmax(200px, 1fr)).

🧠 Task 12: Center All Items in the Grid.

Question: How do you center content both horizontally and vertically inside a grid?

Description: place-items is shorthand for aligning items in both directions.

Hint: Use place-items: center;.
