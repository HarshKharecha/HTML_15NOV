Task 1: Bold and Strong

Question: Write HTML to display the sentence:

ðŸ‘‰ "This word is bold, and this one is important."

  • Use <b> for "bold"
  • Use <strong> for "important"
Answer: 

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Bold and Strong</title>
        </head>
        <body>
              <!--Ans-->
              <p><strong>Output:</strong> This word is <b>bold</b>, and this one is <strong>important</strong>.</p>
        </body>
        </html>

Output: This word is bold, and this one is important.
