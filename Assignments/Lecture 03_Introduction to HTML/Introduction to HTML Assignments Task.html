HTML Assignments Question & Answer

Task 1: Define HTML

Question: What does HTML stand for and what is its purpose?

Answer:
  • HTML stands for Hypertext Markup Language.
  • Purpose of HTML: It is the basic standard language for defining the structure and content of documents displayed on the World Wide Web.

Task 2: Identify HTML’s Role

Question: Why is HTML called the backbone of a webpage?

Answer:
  • HTML is called the basis of web pages because it is the basic structure of the World Wide Web.
  • Headings, paragraphs, images, links, objects etc. are displayed in a basic structure with tags.
  • It allows linking of CSS file (for design) and JS file (for logic).
  • If there is no HTML file, then CSS file and JS file cannot be used.

Task 3: List Reasons Why HTML is Important

Question: Give 3 reasons why HTML is important.

Answer:
  1. Backbone of Any Webpage: HTML forms the basic structure of the World Wide Web.
  2. Compatible Across All Devices and Browsers: HTML is compatible with PC, Laptop, and Mobile Screens (Responsive Design) and works with browsers like Chrome, Microsoft Edge, and Safari (Cross-Browser Consistency).
  3. Essential for SEO: HTML is crucial for Search Engine Optimization (SEO) as it helps search engines index the content properly, improving the visibility of web pages.

Task 4: Match HTML Version with Feature

Question: Match the HTML version with its feature.

Answer:
  • HTML 1.0 (1991): The first version, designed for basic text formatting and links.
  • HTML 2.0 (1995): Standardized features like forms and tables.
  • HTML 3.2 & 4.01 (1997–1999): Introduced styles, scripting support, and multimedia.
  • XHTML (2000): A stricter, XML-based version of HTML.
  • HTML5 (2014): Modern standard supporting multimedia, semantic elements, APIs, and responsive design.

Task 5: Why HTML5 is Important?

Question: Write 2 reasons HTML5 is widely used.

Answer:
  • Multimedia Support: HTML5 offers native support for audio and video elements (<audio> and <video>), eliminating the need for third-party plugins like Flash.
  • Semantic Elements & Responsiveness: HTML5 introduced semantic elements (e.g., <header>, <footer>) for better structure and APIs that enable responsive design and complex web applications.

Task 6: Real-World Applications

Question: Name 3 real-world uses of HTML.

Answer:
  1. Website/Web Application Structure: HTML defines the text, images, links, and layout of every page you see on the World Wide Web.
  2. Internet Navigation (Hyperlinks): HTML's hypertext functionality (the <a> tag) is what allows you to click a link and navigate from one page to another.
  3. Client-Side Data Entry (Forms): HTML forms (like login boxes, contact forms, and survey fields) are used to collect data from users for online applications.

Task 7: Identify HTML Structure

Question: What is the role of <head> and <body> in an HTML document?

Answer:
  • <head>: Contains meta-information like the title, character set, and linked resources.
  • <body>: Contains the content displayed on the webpage, such as text, images, and links.

Task 9: Create First HTML File

Question: Write the steps to create your first HTML file.

Answer:
  1. Open a Text Editor: Use Notepad, Notepad++, or VS Code (any text editor).
  2. Create a Folder and Write the Basic HTML Structure: Create a folder (e.g., "HTML") and save a new file named first.html in it.

    Then, enter the following HTML code: 
                    <!DOCTYPE html>
                <html>
                    <head>
                        <title>My First Webpage</title>
                    </head>
                    <body>
                        <h1>Hello, World!</h1>
                        <p>This is my first paragraph of HTML.</p>
                    </body>
                </html>
  3. Save the File: Press Ctrl + S to save the file.
  4. View the File in a Browser: Locate the first.html file on your computer and double-click to open it in your default web browser.

Task 13: SEO and HTML

Question: How does HTML help in SEO?

Answer:

HTML helps search engines understand and rank your webpage:

  • Titles and Descriptions: Tags like <title> and <meta> tell search engines what the page is about and what to show in search results.
  • Headings: Tags like <h1> and <h2> structure content, highlighting the most important topics.
  • Image Descriptions: The alt attribute for images helps search engines understand the content of pictures.
  • Clear Structure: Using modern HTML tags makes your site accessible and easier for Google to process, boosting your rank.
