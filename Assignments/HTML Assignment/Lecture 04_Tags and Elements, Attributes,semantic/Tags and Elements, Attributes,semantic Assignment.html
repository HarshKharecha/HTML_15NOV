Task 1: Define Tag vs Element

Question: What is the difference between a tag and an element in HTML?

Answer:
  • A tag is just the name enclosed in angle brackets (e.g., <p>), whereas an element consists of an opening tag, content, and a closing tag (e.g., <p>Hello<p>).

Task 9: <main> Element

Question: What is the role of the <main> tag?

Answer:
  1. The <main> tag holds the main and most important content of the webpage.
  2. It contains the most important information that the page is about.
  3. It does not include things like the header, footer, or sidebar.
  4. It helps browsers and screen readers easily find the main part of the page.
  5. It also makes the webpage more organized and clear.
