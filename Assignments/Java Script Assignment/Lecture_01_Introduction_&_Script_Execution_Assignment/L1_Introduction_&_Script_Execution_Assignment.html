🟢 Beginner Level (Foundations)

1. First Script Execution

Task:

  • Write a JavaScript file that logs three messages.
  • Observe execution order in the console.

Goal:

Understand top-to-bottom synchronous execution.

Answer:

1. First Script Execution

JavaScript runs line by line from top to bottom. This is called synchronous execution.

👉 That means:

  • First line runs first
  • Second line runs next
  • No skipping or jumping

💡 Think of it like reading a book 📖 — one line at a time

2. Inline vs External Script

Task:

  • Add one script inline in HTML and one external JS file.
  • Log messages from both.

Goal:

Learn how scripts are loaded and executed.

Answer:

2. Inline vs External Script

There are two ways to write JavaScript:

🔹 Inline Script

  • Written inside HTML file (<script> tag).
  • Executes immediately where it is placed.

🔹 External Script

  • Written in a separate .js file.
  • Loaded into HTML using <script src="">.

👉 Execution depends on where the script is placed, not just type.
